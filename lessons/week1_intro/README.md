# Tuần 1: Chào bạn, mình là Python!

Chào mừng bạn đến với thế giới lập trình kỳ diệu! Hôm nay, chúng ta sẽ làm quen với một "người bạn" siêu thông minh và vô cùng nổi tiếng: **Python**!

Python giống như một chú rắn nhỏ đáng yêu, biết làm rất nhiều trò ảo thuật. Bạn chỉ cần viết hướng dẫn, Python sẽ thực hiện ngay lập tức!

---

## 🌟 Mục tiêu cần đạt
* Hiểu lập trình là gì (viết hướng dẫn cho máy tính).
* Biết cách bắt Python nói chuyện bằng lệnh `print()`.
* Chạy chương trình Python đầu tiên của riêng mình!

---

## 🔑 Khái niệm cốt lõi: Lệnh `print()` là gì?
Hãy tưởng tượng Python là một chú vẹt thông minh. Khi bạn nói:
```python
print("Xin chào thế giới!")
```
Chú vẹt Python sẽ ngay lập tức hét to lên màn hình: `Xin chào thế giới!`

### 📝 Cú pháp rất đơn giản:
1. Viết chữ `print` (viết thường hết nhé, nếu viết hoa `Print` là chú vẹt không hiểu đâu!).
2. Mở ngoặc đơn `(`.
3. Dùng dấu nháy kép `"` hoặc nháy đơn `'` để bao bọc lời nói. Nếu không có dấu nháy, Python sẽ tưởng đó là lệnh và bị bối rối đấy!
4. Đóng nháy kép `"` và đóng ngoặc đơn `)`.

> [!NOTE]
> **Mẹo nhỏ**: Bạn có thể in ra nhiều dòng liên tiếp bằng cách viết nhiều lệnh `print()` xếp hàng dọc!
