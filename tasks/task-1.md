# Task 1-1: Xây dựng trang web hướng dẫn lập trình Python căn bản cho trẻ từ 12 tuổi

Xây dựng trang web hướng dẫn lập trình Python căn bản cho trẻ từ 12 tuổi
- Frontend: sử dụng Vite, React, Bootstrap
- Backend: nếu cần thiết thì sử dụng FastAPI (Python)
- Giao diện chia thành 2 cột chính, cột bên trái là lý thuyết, cột bên phải là code mẫu cho trẻ thực hành
- Trình bày giao diện sao cho bắt mắt, tươi tắn, thu hút sự quan tâm của trẻ em
- Xây dựng bài giảng lập trình Python căn bản chia theo lộ trình từng tuần, mỗi tuần chỉ rõ: mục tiêu cần đạt, khái niệm cốt lõi (như biến, vòng lặp, điều kiện, hàm, lớp...), và một dự án thực hành nhỏ, vui nhộn, yêu cầu ngôn ngữ diễn đạt đơn giản, gần gũi và không dùng thuật ngữ quá nặng nề.
  - Bài giảng và code mẫu được tổ chức lưu theo từng nhóm thư mục rõ ràng, bài giảng lưu định dạng markdown
